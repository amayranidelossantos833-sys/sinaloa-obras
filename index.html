⚙️ La app necesita conectarse a Firebase para guardar datos compartidos. Sigue los pasos de abajo — solo se hace una vez.
Guía de configuración — sin programar
1

Crea una cuenta de Google (gratis)

Si ya tienes Gmail, usa esa cuenta. Ve a firebase.google.com e inicia sesión.

2

Crea un proyecto Firebase

Clic en "Agregar proyecto" → ponle nombre (ej. sinaloa-obras) → desactiva Google Analytics → Crear proyecto.

3

Activa Realtime Database

En el menú izquierdo: Compilación → Realtime Database → Crear base de datos. Elige la ubicación más cercana (us-central1) y selecciona Modo de prueba (puedes ajustar reglas después).

4

Obtén tus credenciales

Ve a Configuración del proyecto ⚙️ → General → Tus apps → Agrega app web (</>). Copia el objeto firebaseConfig que aparece.

5

Edita el archivo index.html

Abre index.html en el Bloc de Notas. Busca el texto REEMPLAZA_CON_TU_API_KEY y sustituye todo el bloque firebaseConfig con tus datos reales.

const firebaseConfig = { apiKey: "AIzaSy...", authDomain: "mi-proyecto.firebaseapp.com", databaseURL: "https://mi-proyecto-default-rtdb.firebaseio.com", projectId: "mi-proyecto", storageBucket: "mi-proyecto.appspot.com", messagingSenderId: "123456789", appId: "1:123:web:abc123" };
6

Publica gratis en GitHub Pages

1. Crea cuenta en github.com
2. Clic en + New repository → nombre: sinaloa-obras → Public → Create
3. Sube los 3 archivos (index.html, manifest.json, sw.js)
4. Ve a Settings → Pages → Branch: main → Save
5. Tu URL quedará: https://tu-usuario.github.io/sinaloa-obras

7

Instala en el celular (iOS y Android)

iPhone: Abre la URL en Safari → ícono compartir (cuadro con flecha) → "Agregar a pantalla de inicio"
Android: Abre en Chrome → menú ⋮ → "Instalar app" o "Agregar a pantalla de inicio"

✅ ¡Listo! Comparte la URL con tus 2 compañeros y todos verán los mismos datos en tiempo real desde cualquier dispositivo.
